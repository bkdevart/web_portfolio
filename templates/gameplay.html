How this Dashboard Works

The data used to create this dashboard consists of two tables: one is a log of time spent by day on each game, and the other is a record of which games have been completed. Both are CSV files.

game_log.csv

There are four fields to this table: title, time_played, date, and system. {{game_log | safe}}

completed.csv

There are three fields to this table: title, complete, date_completed. {{completed | safe}}

Processing the Data

These tables are imported using the Python library pandas, which will also be used to perform calculations on the data. Fields are created that will be used frequently in analysis, such as day of week, cumulative time spent, and the start date of each week for a given data sample. {{source_sample | safe}}
